## 一、整体架构

```
你的电脑 (Windows 11)                          云服务
┌──────────────────────────┐       ┌──────────────────────────────┐
│                          │       │  阿里云 PolarDB Supabase      │
│  前端 Vue 3              │ HTTP  │  PostgreSQL 17                │
│  http://localhost:5173   │──────→│  杭州可用区J, <30ms延迟        │
│                          │       │  数据库: postgres             │
│  后端 FastAPI            │       │  用户: postgres               │
│  http://localhost:8000   │       └──────────────────────────────┘
│                          │
│  Celery Worker           │       ┌──────────────────────────────┐
│  (异步视频生成)            │ Redis │  Upstash Redis                │
│                          │──────→│  新加坡, 免费300MB             │
└──────────┬───────────────┘       └──────────────────────────────┘
           │
           │ API 调用
           │
┌──────────▼───────────────┐
│  腾讯云 MPS (TTS语音)      │  声音克隆 + 语音合成
│  火山引擎 Seedance (视频)  │  数字人口播视频生成
│  阿里云 OSS (文件存储)     │  音频/视频/图片存储
└──────────────────────────┘
```


## 九、视频生成流程

```
用户提交
  ↓
① 扣减算粒
  ↓
② TTS 语音合成（腾讯云 SyncDubbing）
  ↓
③ 音频上传 OSS
  ↓
④ Seedance 视频生成（火山引擎方舟）
  │  - 人物提示词（AI脱口秀）或 参考图片（数字人）
  │  - TTS 音频
  │  - 参考视频 URL（可选，用于模仿风格）
  ↓
⑤ 轮询等待生成完成
  ↓
⑥ 视频 URL 存入数据库
  ↓
⑦ 用户下载/查看
```
